Como navegar (passo a passo)

  1. Use Legendas Gerais para filtrar tudo: período, origem, produto, AS/DS, contrato, NFSe e status.
  2. Confira os KPIs no topo (bruto, líquido, ticket e volumes por grupo).
  3. Abra o grupo desejado (Greenn / Hotmart / Itaú / Acomp.) e refine com a busca da própria seção.
  4. Use Colunas para mostrar/ocultar campos conforme o objetivo (financeiro, comercial, conciliação).
  5. Em células longas, use “ver mais / recolher” (limite visual de até 3 linhas).
  6. Use Casos / Jornada para entender a trilha comercial-financeira por pessoa.

Leitura correta dos grupos

  • Greenn / Hotmart: vendas/transações.
  • Itaú Extrato: movimentações bancárias efetivadas.
  • Boletos Itaú: carteira (pago / a vencer / vencido).
  • Acomp. Comercial: controle operacional (não é fechamento financeiro final).
  • Casos / Jornada: camada analítica (não altera bases linha a linha).
